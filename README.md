# Radix-64
## 내용
Radix-64 인코딩/디코딩 프로그램.

## 사용법
예시 1
`> ra.exe Test입니다.`

결과 1
`VGVzdMDUtM+02S4=`
`Test입니다.`

예시 2
`> fra.exe 파일경로`

결과 2
encoding.txt, decoding.bin 파일 생성

| File name | Description |
